## About

datsforge is an indie software and game development studio focused on building practical software and games with care and intent.

Software is built to solve real problems, and games are built to be worth playing. Projects are designed to remain understandable, maintainable, and adaptable as they grow.

---

## What gets built here

### Game Development
Indie games built in Java/Kotlin using [libGDX](https://libgdx.com/)  
Pixel art and animations created from scratch

### Android Development
Java/Kotlin applications with straightforward interfaces  
Built for usability and clarity

### Systems & Tools
Utilities, automation, and workflow tools  
Built to improve or simplify everyday tasks

---

## Standards

- Focus on real problems over abstract ones
- Respect the user’s time in design and behavior
- Keep systems understandable by default
- Keep intent and behavior transparent
- Design for long-term maintainability
- Add complexity only when it is required
