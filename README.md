## About

datsforge is an indie software and game development studio focused on building practical software and games, forging ideas into reality with care and persistence.

Software is built to solve real problems. Games are built to be worth playing. Projects are built so they stay understandable, maintainable, and adaptable as they grow over time.

(despite the fancy logos and slogans, yes, dat is just me)

---

## What gets built here

### Game Development

Indie games built in Java/Kotlin using [libGDX](https://libgdx.com/)  
Pixel art and animations created from scratch

### Android Development

Java/Kotlin applications with straightforward interfaces  
Built for usability and clarity

### Systems & Tools

Utilities, automation, and workflow tools  
Built to improve or simplify everyday tasks

---

## How things get shaped

- Respect the user’s time in how things behave  
- Focus on real problems over abstract ones  
- Keep systems understandable (so I don't curse myself six months later)  
- Keep intent and behavior transparent  
- Design for long-term maintainability  
- Add complexity only when the simpler version already failed
